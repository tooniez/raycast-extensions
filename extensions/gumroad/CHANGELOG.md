# Gumroad Sales Changelog

## [Enhancements] - 2026-03-04

- Added price filter (All Sales, Hide $0 Sales) to View Sales command.
- Added keyboard shortcut (Cmd + Shift + H) to toggle $0 sales visibility.
- Added Copy Product URL action to View Products.
- Added Copy Customer Email action to View Sales.
- Refined UI with consistent secondary text icon coloring.

## [OAuth Support] - 2024-11-25

Use OAuth via Raycast's OAuth Proxy to log in OR continue to use Access Token

## [Initial Version] - 2024-11-17

Initial version of the Gumroad Sales extension
