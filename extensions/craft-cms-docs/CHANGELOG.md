# Craft CMS Changelog

## [Add Plugin Store Search] - 2026-03-26

### Added

- New **Search Plugins** command to browse and search the Craft CMS plugin store
- Grid view with plugin icons, paid/free indicators, and developer names
- Detail view with full plugin metadata: pricing, editions, active installs, cloud compatibility, categories, keywords, and links
- Filter plugins by type (Free, Paid, Cloud Compatible) or category
- Copy shell and DDEV install commands for any plugin
- Full-text search across all plugins
