# Music News

Read the latest music news from various sources directly in Raycast.

## Features

- **Latest News**: View the latest articles from multiple music news outlets.
- **Filter by Source**: Quickly filter news by sources like NME, Rolling Stone, Billboard, and more.
- **Open in Browser**: Open articles directly in your browser.
- **Copy URL**: Easily copy the article URL to your clipboard.

## Sources Included

- Kerrang!
- NME
- Billboard
- Loudwire
- Uncut
- Rolling Stone
- Pitchfork
- Louder Than War

## Getting Started

1. Install dependencies: `npm install`
2. Run in development: `npm run dev`
3. Build for distribution: `npm run build`
