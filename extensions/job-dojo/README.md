# Job Dojo

AI-powered interview assistant for Raycast. Ask questions, generate recruiter messages, and create LinkedIn connection templates.

## Setup

1. Create an account at [jobdojo.app](https://jobdojo.app)
2. Go to Settings and generate an API key
3. Open this extension's preferences in Raycast and paste your API key

## Commands

### Ask Job Dojo

Ask questions and get AI-powered answers to help with your job search and interview preparation.

### Generate Recruiter Message

Paste a job description to generate a professional recruiter outreach message.

### Generate Connection Request

Paste a job description to generate a LinkedIn connection request message.

### Check Status

View your Job Dojo account status and usage.

### Add Job Application

Add a new job application to track your job search.

## Preferences

- **API Key** (required): Your Job Dojo API key from jobdojo.app/settings
- **API Base URL**: The base URL for the API (default: https://jobdojo.app)
