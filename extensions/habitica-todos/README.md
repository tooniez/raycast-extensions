# Habitica Raycast Extension

Manage Habitica ToDo's from Raycast

## Setup

This extension requires your Habitica User ID and API Key to function.

### Getting Your Credentials

1. Log in to [Habitica](https://habitica.com)
2. Go to **Settings** → **Site Data**
3. Copy your **User ID** and **API Key**
4. Paste them into the extension preferences when prompted

**Note:** Keep your API Key private. Do not share it publicly.

## Features

* View and manage your Tasks
