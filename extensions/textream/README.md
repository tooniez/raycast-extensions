# Textream

Raycast extension for [Textream](https://blog.fka.dev/textream/) — a free macOS teleprompter app for streamers and presenters.

[Website](https://blog.fka.dev/textream/) • [GitHub](https://github.com/f/textream)

## Commands

- **Read with Textream**: Send text to Textream's teleprompter overlay. Prefills from selected text or clipboard.
- **History**: View and reuse previously sent texts.
