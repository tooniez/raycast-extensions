# Raycall Changelog

## [Initial Release] - 2026-06-12

- Save any URL from Raycast with one keystroke (auto-detects active browser tab or clipboard)
- Semantic search across saved bookmarks using vector embeddings + LLM rerank
- Confidence-coded score badges on search results
- Surfaces in-progress indexing in a dedicated section so newly-saved URLs are visible immediately
- Background indexing on the server — Raycast feels instant
