# Nibit Raycast Extension Changelog

## [Initial Release] - 2026-06-02

- Encrypted push inbox with background sync and realtime updates
- Send encrypted text, URLs, and files to your Nibit devices
- Send Push arguments prefill the form and require confirmation before sending
- Latest Push commands: copy, paste, and open the newest push instantly
- Low-latency cache with bounded freshness verification
- End-to-end encryption using NaCl/TweetNaCl
- OAuth PKCE authentication via Nibit auth bridge
