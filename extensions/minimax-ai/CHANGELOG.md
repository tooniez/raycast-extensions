# Changelog

## [1.1.0] - 2026-02-14

- Add MiniMax M2.5 model support (now default)
- Remove non-existent highspeed model variants
- Update model selection: M2.5 (recommended), M2.1, M2

## [Initial Release] - 2026-02-09

- Add AI Chat command with conversation history
- Add Ask AI command for quick questions
- Streaming responses support
- Multiple model selection (MiniMax-M2.1, MiniMax-M2)
- Configurable temperature and max tokens
- Concise mode for shorter responses
- Custom system prompts
