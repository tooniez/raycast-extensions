# Privacy

## Data Sent to Zo

- Prompt content and selected model for Zo chat requests.
- Auth headers containing the configured Zo API key.

## Data Stored Locally

- Default selected model.
- Activity records (tool/API metadata and redacted parameters).

## Data Not Stored

- Plaintext secrets in activity records.
- API keys outside Raycast secure preference storage.

## Notes

- Users should avoid placing secrets in free-form prompts where possible.
