# Raycast Store Metadata Draft

## Extension

- Title: Zo
- Description: Use Zo API chat and models from Raycast.
- Categories: Developer Tools, Productivity
- Keywords: zo, ai, chat, models, productivity

## Commands

- Zo Setup: Validate credentials and check Zo API connectivity.
- Zo Chat: Chat with Zo models with optional streaming (off by default) and hidden-by-default thinking output.
- Zo Models: Manage model defaults.
- Zo Activity: Review and replay prior chat runs.

## Review Notes

- External network calls go only to configured Zo API endpoints.
- API keys are read from Raycast preferences and redacted from local activity records.
