# Xbox Friends Changelog

## [Initial Version] - 2026-03-12

- Initial release of Xbox Friends extension
- View Xbox Live friends grouped by Online, Away, and Offline status
- See what game friends are currently playing
- Last seen time for offline friends
- Broadcasting indicator for friends who are live streaming
- Open a friend's Xbox profile in the browser
- Copy gamertag to clipboard
- Manual refresh with ⌘R
