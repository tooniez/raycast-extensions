# Launchd Monitor Changelog

## [Initial Version] - 2026-05-18

- Menu bar display showing launchd job status at a glance
- Monitor multiple launchd jobs via comma-separated labels
- View last exit status, run times, and schedule info
- Detects currently-running jobs via PID
- Supports `StartCalendarInterval` and `StartInterval` schedules
- Re-run jobs on demand from the menu bar (system daemons prompt for admin password)
- View recent log output for each job
