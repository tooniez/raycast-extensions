# e18e Module Replacements

Browse and search the [e18e](https://e18e.dev) community's curated list of modern replacements for outdated npm packages — right from Raycast.

## Features

- **Search** across 700+ packages instantly
- **Alphabetical sections** for quick browsing
- **Detail view** per package with type, description, code snippets, and links
- **Copy snippets** directly to clipboard for simple drop-in replacements
- **Open docs** on MDN, Node.js, or e18e with one keystroke

## Replacement types

| Type                | Meaning                                     |
| ------------------- | ------------------------------------------- |
| 🟢 Native           | Built into the platform — no package needed |
| 🔵 Simple           | A one-liner or small snippet replaces it    |
| 🟡 Community choice | A well-known, more performant alternative   |
| 🔴 Removal          | The package can simply be removed           |

## Data source

Powered by [module-replacements](https://github.com/es-tooling/module-replacements), maintained by the e18e community.
