# DevContainer Features Changelog

## [Initial Version] - 2025-02-19

- Initial release of DevContainer Features extension
- Browse all DevContainer features from official collection index
- Search features by name or description
- View detailed documentation with README and options
- Copy feature references for devcontainer.json
- Generate configuration snippets with all options
- Mark favorite features for quick access
- Filter by official/third-party collections
- Configurable caching (1 hour to 7 days)
- Optional GitHub token for higher API rate limits
