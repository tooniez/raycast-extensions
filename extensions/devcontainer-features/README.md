# DevContainer Features

Search and browse DevContainer features from the official registry.

## Features

- Browse all DevContainer features from the official collection index
- Search features by name or description
- View detailed feature documentation including README and options
- Copy feature references for use in devcontainer.json
- Generate configuration snippets with options
- Mark favorite features for quick access
- Filter by official/third-party collections

## Usage

1. Open Raycast and search for "DevContainer Features"
2. Browse or search for a feature
3. Press Enter to view feature details
4. Use actions to copy reference or configuration

## Preferences

- **GitHub Personal Access Token**: Optional token to increase API rate limit from 60 to 5000 requests/hour
- **Cache Duration**: How long to cache feature data (1 hour to 7 days)
- **Fetch Concurrency**: Number of parallel API requests (5-20)

## Data Sources

Features are fetched from:
- [DevContainers Collection Index](https://github.com/devcontainers/devcontainers.github.io)
- [GitHub Container Registry (GHCR)](https://ghcr.io)
