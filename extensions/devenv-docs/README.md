# DevEnv Docs

Browse and search the [DevEnv](https://devenv.sh) documentation directly from Raycast.

## Features

- Browse the DevEnv Guide documentation
- Search through documentation pages and options
- View language and service configurations with their types and default values
- Quick access to the official devenv.sh website

## Usage

Simply open Raycast and search for "DevEnv Docs" to start browsing the documentation. Use the search bar to filter through available documentation pages and options.
