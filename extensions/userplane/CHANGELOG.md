# Userplane Changelog

## [Initial Version] - 2026-05-07

- Create recording links with workspace, project, domain, and reference selection.
- Browse recordings in a thumbnail grid with filters (creator, project, link), sorting, and expiry indicators.
- Browse recording links with filters (creator, project, domain) and sorting.
- Menu bar command with quick-launch shortcuts to every command and the Userplane dashboard.
- Per-workspace filter persistence and last-used workspace/domain/project memory.
- Support for self-hosted and staging deployments via API Host and Dashboard Host preferences.
