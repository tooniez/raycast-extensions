# Plex Changelog

## [Design Overhaul] - 2026-03-09

- Richer detail view with genres, cast, directors, writers, studio, and content rating
- Replaced posters in detail view with art to better fit the layout
- Deep links to Plex and search links for IMDB and TMDB
- Optimized image loading by using Plex native URL transformation
- Watch status visibility (unwatched, in progress, watched) in grid and detail
- Context-aware subtitles in grid that change based on active sort
- New sort options: critic/audience rating, duration, last watched, random
- Persistent default sort preference with "Set as Default" action
- Standardized utility files: `.ts` extensions, named exports, one concern per file
