# Open Latest URL from Clipboard

Opens the latest URL from Clipboard if one is found in the latest 6 entries

## Features

- Opens the latest URL from your clipboard history
- Configurable protocol support:
  - HTTP
  - HTTPS
  - FTP
  - File
  - Mailto
  - Tel
- Custom URL schemes for app deep links — add your own comma-separated schemes (e.g. `obsidian`, `slack`, `vscode`) in the command preferences to open links like `obsidian://open?vault=Notes`
