# WooCommerce Quicker Changelog

## [Initial Version] - 2026-04-30

- Search and filter WooCommerce orders by status
- Search customers by name, email, or role
- Browse products with pricing, stock status, and SKU
- Multi-store management with credential validation
- Per-store currency formatting
- Local/self-signed SSL support for development stores
- Direct links to WooCommerce admin for all items
