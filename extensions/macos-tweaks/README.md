# macOS Tweaks

</div>

Discover and toggle hidden macOS settings without the Terminal. Browse a curated catalog of `defaults write` tweaks for Finder, Dock, Screenshots, Animations, Sound, and more.

## Commands

- **Browse Tweaks** — Browse all hidden settings by category, toggle them on/off, and see current values at a glance
- **My Tweaks** — View only your modified settings, reset individually or all at once, export as shell commands
- **Tweaks Menu Bar** — Quick access from the menu bar with badge showing how many tweaks are active

## Features

- 108 curated tweaks across 14 categories
- Filter by category or status (All / Modified / Default)
- One-click toggle for boolean settings
- Dropdown selection for enum settings (view style, screenshot format, minimize effect, etc.)
- Copy `defaults write` command to clipboard for any tweak
- Reset individual tweaks or all at once to system defaults
- Export all modified tweaks as a shell script
- Safety warnings for moderate-risk settings
- Automatic process restart (Finder, Dock, SystemUIServer) after applying changes
- Quick toggle directly from the menu bar
