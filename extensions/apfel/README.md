# Features

## Ask anything

Straight from your command bar, ask anything you want answered.

![Ask anything](metadata/chat.png)

## Custom system prompts

Customize the model to your liking.

![Custom models](metadata/models.png)

## Conversations

Access your conversations from conversation history.

![Conversations](metadata/conversations.png)

## Translate

Translation between supported languages.

![Translate](metadata/translate.png)

## Explain files & folders

Explain local files & folders.

![Explain files](metadata/explain-file.png)

✨ All without any logins, limits, payments or even a network connection!

# Requirements

- Apple Silicon Mac
- macOS Tahoe (26.0) or higher
- Apple Intelligence enabled
- Apfel CLI from Homebrew or [source](https://github.com/Arthur-Ficial/apfel)
  - ```bash
      brew install apfel
    ```
