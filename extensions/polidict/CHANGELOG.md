# Changelog

## [Initial Version] - 2025-01-27

- Lookup Word: Search and add vocabulary with definitions from blueprints or AI
- View Vocabulary: Browse, filter, and edit saved words
- Start Training: Practice with Quiz, Quiz Reverse, and Flashcard modes
- Quick Add Word: Save a word from typed text or current selection
- Manage Groups: Create, edit, and delete vocabulary groups
- View Stats: View learning statistics, streak, and training history
- Raycast AI support for Pro subscribers (generates definitions when no other source available)
