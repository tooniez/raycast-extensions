## Arcane Wallpaper

Browse Arcane wallpapers and set them as desktop wallpaper from Raycast.

Wallpapers are loaded from a curated Google Drive folder allowlist in `src/wallpapers.ts`.

## Disclaimer

This extension is fan-made and is not affiliated with, endorsed by, or sponsored
by Riot Games, Netflix, or the Arcane series.

**Tips:**

- If you don't like the automatic wallpaper switching feature, turn it off by
turning off the Background Refresh function in the command's preferences
