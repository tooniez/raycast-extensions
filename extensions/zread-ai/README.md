# zread.ai

Open `zread.ai` pages for GitHub repositories from Raycast.

## Commands

### Open zread.ai Page

Accepts any of these inputs:

- GitHub repository URL, like `https://github.com/instructkr/claw-code`
- `owner/repo`, like `instructkr/claw-code`

If you do not provide an argument, the command falls back to the current browser tab URL when the Raycast Browser Extension is installed.

### Trending Repositories

Opens the public trending page on `zread.ai` in your browser.
