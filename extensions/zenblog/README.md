# Zenblog

This is a Raycast extension for [Zenblog](https://www.zenblog.com/) - _A tiny blogging CMS_.

## ⚙️ Configuration

For each blog you want to add to the extension:
1. `Run` the command
2. In Zenblog, `Navigate` to Blog **Settings**
3. `Copy` "Blog ID"
4. `Paste` in **Form**
