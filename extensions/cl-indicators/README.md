# Chilean Indicators

View Chilean financial indicators (Dólar, UF, Euro, UTM) in Raycast. Values come from the public [mindicador.cl](https://mindicador.cl/) API.

## Features

- Lists all four indicators with formatted CLP amounts
- Shows the last update date per series
- Copy a value from the action panel or refresh the list

## Usage

1. Open Raycast and run **View Chilean Indicators**.
2. Select a row and use **Copy Value**, or press **⌘ R** to refresh.

## Development

```bash
npm install
npm run dev
```

Before opening a pull request, run `npm run build` and `npm run lint`.
