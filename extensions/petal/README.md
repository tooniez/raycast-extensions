# Petal

Control the Petal macOS app from Raycast.

## Commands

- Start Recording
- Stop Recording
- Search History
- Switch Model
- Copy Last Transcription

## Requirements

- Petal must be installed: https://github.com/Aayush9029/petal/releases/latest
- Petal history defaults to `~/Documents/petal/history/history.json`

## Preferences

- History Directory
- Models Directory
