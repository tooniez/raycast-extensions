# Gift Stardew Valley

## Description

This Raycast plugin helps you quickly find the best gifts for the characters in Stardew Valley. It provides an easy-to-use interface to search for characters and see what they love, like, dislike, and hate.

## Features

- Search for characters by name or by gift name
- View loves, likes, dislikes, and hates for each character
- Quick access to gift information with images

## Development

```bash
npm install
npm run build
npm run lint
```

If `npm run lint` reports that ESLint or Prettier is not installed, install with dev dependencies: `npm install --include=dev`.
