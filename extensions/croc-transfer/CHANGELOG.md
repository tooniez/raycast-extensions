# Croc Transfer Changelog

## [Initial Release] - 2026-05-26

### Added
- **Send File** command: Pick files or folders (or pre-select in Finder), optionally set a custom code phrase, and watch live transfer progress with speed, ETA, and size stats
- **Quick Send** command: Select files in Finder, invoke this command, and the code phrase is copied to your clipboard immediately — no window required
- **Receive File** command: Enter a code phrase to receive files. Automatically pre-fills from your clipboard. Supports Raycast deep links so senders can share a one-click receive link
- **Transfer History** command: Browse past sends and receives grouped by date. Preview images and text files inline, re-send previous files, reveal received files in Finder, and copy code phrases or deep links at any time
- End-to-end encryption via croc for all transfers
- Support for custom relay servers
- Configurable download directory
- Real-time progress display with speed, ETA, and size stats
