# Cut Out Changelog

## [Documentation Update] - 2026-03-03

- Updated the README to clarify that Xcode (or Xcode Command Line Tools) is required to compile the Swift helper on first launch.

## [Initial Release] - 2026-03-03

- Added a visual Cut Out editor to remove horizontal or vertical sections from images.
- Added `Cut out Image Section from Selection` for images selected in Finder.
- Added `Cut out Image Section from Clipboard` for clipboard-based image editing.
- Added automatic stitching after section removal so the final image stays clean.
