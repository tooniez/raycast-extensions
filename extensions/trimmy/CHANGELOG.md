# Trimmy Changelog

## [Initial Release] - 2026-03-30

### Added

- `Trim and Copy` Raycast command for on-demand cleaning
- `Trim and Paste` command to clean and immediately paste into the frontmost app
- `Trim and Preview` command to inspect the result before copying or pasting
- Aggressiveness preference with Low, Normal, and High modes
- Default-on selected text / Finder item fallback before clipboard
- Command-aware multiline flattening based on Trimmy's `TextCleaner` heuristics
- Prompt stripping, wrapped URL repair, path quoting, and box-drawing cleanup
- HUD success feedback and error toasts for failures
