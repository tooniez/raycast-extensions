# Commit Streak

A Raycast menu bar extension that tracks your GitHub contribution streak.

## Features

- 🔥 Shows your current streak count when you've contributed today
- ❄️ Shows how many days you've gone without a contribution
- 🏆 Milestone icons unlock as your streak grows
- Refreshes automatically every 10 minutes

## Setup

1. Generate a **GitHub Personal Access Token** at [github.com/settings/tokens](https://github.com/settings/tokens)
   - Use a **Classic token** with the `read:user` scope
2. Open the extension in Raycast and enter your token and GitHub username in the preferences

## Milestones

Reach certain streak milestones and discover special icons along the way.
