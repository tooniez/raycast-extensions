# colorblind

Simulate colour blindness on an image or colour.

## Inputs

- `INPUT` optional: image file, or colour string when `--colour` is used.

## Options

- `--cb-type <CB_TYPE>`: `normal`, `protanopia`, `deuteranopia`, `tritanopia`, `protanomaly`, `deuteranomaly`, `tritanomaly`, `achromatopsia`, or `achromatomaly`; default `deuteranopia`.
- `--colour`: treat input as a colour string instead of an image path.
- Global: `--json`, `--quiet`, `--output`.

## Output

Simulated image or colour output.
