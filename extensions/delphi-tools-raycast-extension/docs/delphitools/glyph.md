# glyph

Look up Unicode characters by codepoint, name, range, or search term.

## Inputs

- `INPUT` optional: codepoint like `U+0041` or `0x41`, or a single character.

## Options

- `--range <RANGE>`: named Unicode block/range, e.g. `arrows`, `latin`, `greek`.
- `--search <SEARCH>`: search Unicode names.
- `--limit <LIMIT>`: maximum search results; default `50`.
- Global: `--json`, `--quiet`, `--output`.

## Output

Unicode character metadata or search results.
