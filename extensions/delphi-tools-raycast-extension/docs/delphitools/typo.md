# typo

Convert between typographic units.

## Inputs

- `VALUE` required: value with unit, e.g. `12pt`.
- `TARGETS` optional variadic: target units such as `px`, `mm`, or `pc`. Shows all when omitted.

## Options

- `--base <BASE>`: base font size in pixels for `em` and `rem`; default `16`.
- Global: `--json`, `--quiet`, `--output`.

## Output

Converted typographic unit values.
