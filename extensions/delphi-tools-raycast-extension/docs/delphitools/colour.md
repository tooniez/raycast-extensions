# colour

Convert a colour between formats.

## Inputs

- `COLOUR` required: hex, `rgb()`, `hsl()`, `oklch()`, `oklab()`, or CSS colour name.
- `FORMATS` optional: one or more of `hex`, `rgb`, `hsl`, `oklch`, `oklab`, `lab`. Shows all formats when omitted.

## Options

- `--pretty`, `-p`: show a colour card.
- Global: `--json`, `--quiet`, `--output`.

## Output

Converted colour values, optionally structured as JSON.
