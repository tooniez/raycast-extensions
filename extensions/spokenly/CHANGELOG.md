# Spokenly Changelog

## [Initial Version] - 2026-06-09

- Add `Toggle Dictation` — start Spokenly's main prompt via the `spokenly://` URL scheme, with optional global-shortcut fallback.
- Add `Copy Last Transcript` / `Paste Last Transcript` — quick access to the most recent dictation result.
- Add `Search Transcripts` — browse the full dictation history with search, copy/paste, reveal recording in Finder, copy the `.wav` as a file or as a path, and trash entries.
- Add `Transcribe File` — pick an audio or video file and transcribe it through Spokenly's local MCP server. Supports text, Markdown, SRT, VTT, and JSON output, plus optional speaker diarization.
- Add `Open History Folder` and `Open Last Recording` — direct shortcuts into the Spokenly data directory.
- Add `Select Dictation Model` and `Select File Transcription Model` — switch the active models without leaving Raycast.
- Add `Run Quick Command` — list and trigger your Spokenly quick commands.
