# OpenRouter Manager

Raycast extension to view your OpenRouter credits and manage your API keys.

## Features

- View your total credit
- View your credit left
- List your API keys and their metadata
- Create API keys
- Delete API keys
- Update API keys

## How to get the Management Key

- Go to your [OpenRouter dashboard](https://openrouter.ai/settings/preferences)
- Go to Settings > Management Keys
- Create a new key and copy its value. OpenRouter won't show it again.
- Go to the preferences in Raycast (or start any command of the OpenRouter extension)
- Set the Management Key you just copied
