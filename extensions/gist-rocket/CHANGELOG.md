# Gist Rocket Changelog

## [Initial Version] - 2026-06-01

- Publish HTML or Markdown from the clipboard as a live web page.
- Publish from a file or folder, with automatic inlining of local scripts and stylesheets.
- Browse, open, update, rename, and delete published pages from the *My Pages* command.
- Markdown rendering with light, dark, or auto themes.
- Per-page light/dark toggle in the top-right corner; choice is remembered in the viewer's browser via localStorage.
- Server-side syntax highlighting for fenced code blocks via highlight.js (JavaScript, TypeScript, Python, Go, Rust, Java, C#, C++, Swift, Kotlin, Ruby, PHP, SQL, Bash, HTML, CSS, JSON, YAML, Markdown, Objective-C, R, Dart, PowerShell, and more).
- Preferences for host URL, default visibility, default theme, auto-open, and auto-copy.
