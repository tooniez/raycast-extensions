# Git Co-Authors

Quickly select co-authors and copy properly formatted `Co-authored-by` trailers for your Git commits.

![](metadata/git-co-authors-1.png)

## Features

- **Choose Co-Authors** — Search and multi-select co-authors from your saved list. Copy or paste formatted trailers directly into your commit message.
- **Add Author** — Manually add co-authors by name and email.
- **Load Co-Authors from Folder** — Scan a folder for Git repositories and bulk-import co-authors from all of them.
- **Clear All Authors** — Remove all saved co-authors at once.

Co-authors are automatically deduplicated by email and sorted by frecency (frequency + recency).

![](metadata/git-co-authors-2.png)

## Commands

| Command                        | Description                                                  |
| ------------------------------ | ------------------------------------------------------------ |
| Choose Co-Author               | Search, select, and copy co-author lines for your commits    |
| Add Author                     | Add a new co-author by name and email                        |
| Load Co-Authors From Folder    | Scan a folder tree for repos and import all their authors    |
| Clear All Authors              | Delete all saved co-authors                                  |

## Keyboard Shortcuts

### macOS

| Shortcut                   | Action                          |
| -------------------------- | ------------------------------- |
| `Enter`                    | Toggle co-author selection      |
| `⌘` `⏎`                   | Copy selected co-authors        |
| `⌘` `⇧` `⏎`              | Paste into active window        |
| `⌘` `N`                   | Add new author                  |
| `⌘` `E`                   | Edit author                     |
| `⌘` `⌫`                   | Remove author                   |
| `⌘` `⌥` `⌫`              | Clear all authors               |

### Windows

| Shortcut                   | Action                          |
| -------------------------- | ------------------------------- |
| `Enter`                    | Toggle co-author selection      |
| `Ctrl` `Enter`             | Copy selected co-authors        |
| `Ctrl` `Shift` `Enter`     | Paste into active window        |
| `Ctrl` `N`                 | Add new author                  |
| `Ctrl` `E`                 | Edit author                     |
| `Ctrl` `Backspace`         | Remove author                   |
| `Ctrl` `Alt` `Backspace`   | Clear all authors               |
