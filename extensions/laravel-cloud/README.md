# Laravel Cloud

Manage your [Laravel Cloud](https://cloud.laravel.com) applications directly from Raycast.

## Features

- **Search Applications** - Browse and search your Laravel Cloud applications and environments
- **View Deployments** - Monitor recent deployments and their status
- **View Logs** - Stream application logs from your environments
- **Run Commands** - Execute commands on your Laravel Cloud environments
- **Manage Domains** - View and manage domains for your environments
- **Manage Environment Variables** - View and edit environment variables
- **Manage Databases** - Manage database clusters, schemas, and snapshots
- **Manage Caches** - View and manage caches for your applications
- **Manage Buckets** - Manage object storage buckets and keys
- **Manage Background Processes** - View and manage background processes for your instances

## Setup

1. Install the extension from the [Raycast Store](https://www.raycast.com/1weiho/laravel-cloud)
2. Generate an API token at [cloud.laravel.com](https://cloud.laravel.com)
3. Enter your API token when prompted
