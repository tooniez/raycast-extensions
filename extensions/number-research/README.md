# Number Research

Research numbers and discover if they have been searched before on [numberresearch.xyz](https://numberresearch.xyz).

## Commands

### Discover Number

Enter any number to check if it has been discovered before. Shows whether you're the first person to search for it, along with the discovery date and total search count.

### Number Stats

View the current number of the day, total numbers discovered, and the top searched numbers leaderboard.
