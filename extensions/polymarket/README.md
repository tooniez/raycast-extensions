# Polymarket

A Raycast extension for searching, viewing, and analyzing markets and trader profiles on Polymarket.com. View market prices, statistics, trader analytics, and trading data directly from Raycast.

## Commands

| Command               | Description                                                          |
| --------------------- | -------------------------------------------------------------------- |
| **Polymarket Markets** | Search Polymarket markets or view top markets by 24h volume         |
| **Polymarket Traders** | Search Polymarket trader profiles by wallet address or username     |

## Features

### Market Search

- Search for specific markets by keyword or question
- View top markets by 24-hour trading volume
- Real-time search results with instant filtering

### Market Details & Analytics

- Market information including outcomes, prices, and volume
- Price history charts with multiple time intervals (1h, 1d, 1w, 1m, max)
- Market status indicators (Active, Closed, Featured, New, Archived, Restricted)
- Volume statistics with 24-hour and total volume tracking
- Raw market data view with JSON export capability
- Market tags and categorization
- Outcome prices and probabilities
- Direct links to Polymarket.com for detailed trading

### Price Charts

- Price charts rendered using ECharts
- Support for multiple outcomes within a single market
- Time-series data visualization with line charts

### Trader Search & Profiles

- Search trader profiles by wallet address (`0x...`) or username
- Today's top 20 traders displayed by default when no search query is entered
- Detailed PnL analytics across multiple timeframes (Daily, Weekly, Monthly, All-Time)
- Active positions with current value, entry price, outcome, and PnL
- Trade history with closed positions, settlement dates, and realized PnL
- Filter positions by title directly from the search bar
- X (Twitter) badge and direct link for linked profiles
- Open trader's Polymarket profile or X profile from the action panel
