# Changelog

All notable changes to "Raycast WordReference Extension" will be documented in this file.

## [1.1.1] - 2026-06-02

### Fixed

- Prevent crashes when WordReference rejects translation detail requests.
- Show a clear translation error page with probable causes and recovery suggestions.
- Classify common WordReference HTTP errors such as bot protection, rate limits, missing pages, and server issues.

## [1.1.0] - 2023-09-19

### Added

- Option : Automatically use the selected text in the search bar.
- Command arguments "Word" and "Language" for a faster search.

## [1.0.0] - 2023-08-11

### Added

- Initial release of Raycast WordReference Extension.
- Quick and easy translation feature.
- Multilanguage translation [French - Spanish - Portuguese - Italian - German - Dutch - Swedish - Icelandic (They have no enemies) - Russian - Polish - Romamian - Czech - Greek - Turkish - Chinese - Japanese - Korean - Arabic]
- Recent searches features 
- Preferences : Select translation language
