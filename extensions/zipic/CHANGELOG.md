# Zipic Changelog

## [AI Extension and Presets] - 2026-05-26

- Rename extension from `images-compression` to `zipic` to match the brand
- Add support for AI Extension via the `optimize-images` tool, so Raycast AI can drive Zipic with `@zipic`
- Add `Compress Images with Options` command for one-off compression with custom settings
- Add `Compress Images with Preset` command that reads compression presets directly from the Zipic app's settings, so a single source of truth is shared between the app and the extension
- Upgrade dependencies to latest Raycast API (1.104.x) and `@raycast/utils` 2.x
