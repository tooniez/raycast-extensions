# Napkin Raycast Extension

Transform your text into engaging visuals, diagrams, and illustrations instantly using [Napkin AI](https://napkin.ai).

## Key Features

- **Text to Visual**: Instantly convert any text snippet into a professional diagram.
- **Multiple Styles**: Choose from a variety of styles including Colorful, Hand-drawn, Monochrome, and Formal.
- **Visual Type Control**: Specify the type of visual you want (e.g., mindmap, flowchart, timeline).
- **Extraction Modes**:
  - `Auto`: Automatically determine the best extraction method.
  - `Rewrite`: Optimize and rewrite text for better visual representation.
  - `Preserve`: Keep the original text as is.
- **Sort Strategies**:
  - `Relevance`: Order layouts by relevance to your content.
  - `Variation`: Prioritize visual diversity.
  - `Random`: Randomize the order of suggested layouts.
- **Multiple Formats**: Export your visuals as PNG, SVG, or PowerPoint.
- **Color Modes**: Support for Light, Dark, or both modes simultaneously.

## Getting Started

1.  **Get your API Key**: Sign up at [Napkin AI](https://napkin.ai) and generate an API key from your settings.
2.  **Install Extension**: Install this extension in Raycast.
3.  **Configure API Key**: When prompted (or in the extension settings), enter your Napkin API key.
4.  **Save Path**: (Optional) Customize where your generated visuals are saved.

## Usage

1.  Open the **Generate Visual** command.
2.  Paste the text you want to visualize.
3.  Customize your options (Style, Format, Language, Visual Type, etc.).
4.  Submit and wait for your visuals to be generated.
5.  Preview PNG/SVG results or directly save PowerPoint files to your chosen path.
