# Infomaniak

Quick access to Infomaniak's manager pages (Domains, Hostings and Email services)

## Configuring a Personal Access Token

You must use a personal access token :

1. Go to <https://manager.infomaniak.com/v3/70035/ng/profile/user/token/add>
3. Add a "Name" for the token.
4. Select the required scopes:

- `accounts`
- `mail`
- `web`
- `domain`

5. Click "Create token".
6. Input your Manager's password
7. Copy the token in the "Infomaniak Token" field in the extension's preferences.
