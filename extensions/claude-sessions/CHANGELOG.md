# Claude Sessions Changelog

## [Initial Release] - 2026-04-16

### Added

- Browse all Claude Code sessions sorted by last activity
- Chat and Code sections with type filter dropdown
- Open sessions directly with `claude --continue` in a new iTerm tab
- Create new named chat sessions with `⌘N`
- Delete sessions with confirmation (`⌘⌫`)
- Copy path (`⌘⇧C`) and Show in Finder (`⌘⇧F`) actions
- Clean Sessions command to remove ghost entries, empty histories, and orphaned directories
- Support for iTerm2, Terminal.app, Ghostty, and Warp via preferences
