# LobeHub Icons

The Raycast extension to search AI / LLM brand SVG logos and icons from [LobeHub Icons](https://icons.lobehub.com/).

## Search Icons

- Browse and search LobeHub icons
- Search by name, provider, group, or variant
- Copy icon component name to clipboard
- Copy SVG to clipboard
- Paste SVG directly into the active app
- Copy additional SVG / PNG / avatar asset URLs
