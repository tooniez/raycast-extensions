# Wallhaven Changelog

## [Initial Release] - 2026-03-03

### Added

- **Search Wallpapers** command with filters for category, purity, sorting, and top range
- **Top Wallpapers** command with time range selector (1d, 3d, 1w, 1M, 3M, 6M, 1y)
- **My Collections** command to browse personal Wallhaven collections (requires API key + username)
- **Random Wallpaper** no-view command that silently sets a random wallpaper
- **Set on All Desktops** action (default Enter)
- **Set on Current Desktop** action (⇧⌘W)
- **Preview Wallpaper** action (⌘Y) with full metadata sidebar including resolution, file size, category, purity, views, favorites, upload date, and color palette
- **Search Similar** action (⇧⌘S) using Wallhaven's `like:` query
- **Download** action (⌘D) saves to configured download directory
- **Copy Image to Clipboard** action (⇧⌘C)
- **Copy Image URL** action (⌘C)
- **Copy Wallpaper ID** action
- **Copy Color Palette** action (copies all 5 dominant colors as hex)
- **Open in Browser** action (⌘O)
- Pagination support across all grid views with correct end-of-results detection
- Extension preferences for API key, username, and download directory
