# Changelog

## [Initial Release] - 2026-02-09

- Interact with Moonshot AI's Kimi directly from Raycast
- Ask questions and get streaming responses in real-time
- Support for multiple Kimi models (Kimi K2.5, Kimi K2 variants)
- Create and manage custom models with personalized prompts
- Save and organize your favorite answers
- Browse conversation history
- Copy questions and answers to clipboard
- Auto-load selected text from frontmost application
- Full text input mode for longer questions
- Dynamic model retrieval from Moonshot AI API
- Configurable temperature and max token limits
- Built-in prompt library with pre-made templates
