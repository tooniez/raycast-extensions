# GetNote Changelog

## [Add Image Notes and Quota] - 2026-05-27

- Added a Save Image Note command with local image files, Finder selection, and image URL download/upload support
- Added a GetNote API quota AI tool
- Improved async task progress feedback for link and image note saves
- Hardened API retry handling for GetNote rate limits

## [Initial Version] - 2026-04-28

- Added commands for recent notes, note search, link saving, and text-note capture
- Added note detail, browser-opening, and delete flows in the Raycast UI
- Added Raycast AI tools for note count, search, detail lookup, save, delete, tags, and knowledge-base workflows
- Added manual credentials and in-app OAuth device flow support
