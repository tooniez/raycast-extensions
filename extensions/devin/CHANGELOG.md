# Changelog

## [Initial Version] - 2026-04-13

- Added a `Devin` command to browse, search, and manage Devin sessions from Raycast
- Added split-view session details with recent message transcript rendering
- Added actions to open sessions in Devin, send follow-up messages, copy session IDs and URLs, and open pull requests when available
- Added a `Create Session` command to start new Devin sessions from Raycast
- Added optional grouping for `My Sessions` vs `Everyone Else`
- Added favorites and recent-session prioritization
