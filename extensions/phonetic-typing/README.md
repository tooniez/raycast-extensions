# Phonetic Typing

Type in English and instantly transliterate to 25 languages — works just like [Google Input Tools](https://www.google.com/inputtools/).

## Features

- **Word-by-word composition** — type a word, press Space to commit the top suggestion, or press Enter to pick a specific one. Build full sentences one word at a time.
- **25 languages** — supports every language available in Google Input Tools transliteration: Amharic, Arabic, Belarusian, Bengali, Bulgarian, Greek, Gujarati, Hebrew, Hindi, Kannada, Malayalam, Marathi, Nepali, Odia, Persian, Punjabi, Russian, Sanskrit, Serbian, Sinhalese, Tamil, Telugu, Tigrinya, Ukrainian, and Urdu.
- **Live suggestions** — see up to 8 transliteration candidates as you type, ranked by relevance.
- **Language switcher** — change language on the fly from the dropdown without leaving the command.
- **Paste or copy** — press Enter to paste the composed text directly into your active app, or Cmd+Shift+C to copy it to the clipboard.

## How to Use

1. Open Raycast and search for **Phonetic Typing**.
2. Select your target language from the dropdown (or set a default in extension preferences).
3. Start typing a word in English (e.g., `namaste`).
4. Press **Space** to commit the top suggestion, or use arrow keys and **Enter** to pick an alternative.
5. Keep typing more words — the composed sentence builds up in the placeholder.
6. When finished, stop typing. Your composed text appears as a list item. Press **Enter** to paste it into your active app.

## Keyboard Shortcuts

| Shortcut            | Action                                           |
| ------------------- | ------------------------------------------------ |
| Space               | Commit top suggestion                            |
| Enter               | Commit selected suggestion / Paste composed text |
| Cmd+Shift+C         | Copy to clipboard                                |
| Cmd+Z               | Undo last word                                   |
| Cmd+Shift+Backspace | Clear all and start over                         |
