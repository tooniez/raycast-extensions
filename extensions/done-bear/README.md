# Done Bear

Done Bear is a task and project manager. This Raycast extension connects to your Done Bear account so you can browse lists (Today, Inbox, Anytime, Upcoming, Someday), search tasks and projects, create tasks and projects, and see Today in the menu bar.

## Sign in

The extension uses OAuth. The first time you run a command, Raycast prompts you to connect your Done Bear account. If your session expires or you sign in again, the extension uses your new token automatically.
