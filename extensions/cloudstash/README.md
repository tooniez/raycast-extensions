# Cloudstash

Save links to [Cloudstash](https://cloudstash.dev) from anywhere on macOS.

## Setup

1. Install the extension from the Raycast Store
2. Run any command — you'll be prompted to connect your Cloudstash account
3. A browser window opens where you log in and click "Connect to Cloudstash"
4. Done — start saving links

## Commands

### Save to Cloudstash

Paste a URL directly into Raycast to save it. Assign a hotkey for fastest access.

### Save Clipboard URL

Saves the URL currently in your clipboard. Ideal for a global hotkey — copy a link anywhere, press the hotkey, done.

## Preferences

| Preference | Description | Default |
|------------|-------------|---------|
| Server URL | Your Cloudstash server URL | `https://cloudstash.dev` |

## Disconnect

To disconnect your account, open Raycast Preferences > Extensions > Cloudstash and click "Disconnect Cloudstash".
