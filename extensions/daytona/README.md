# Daytona Extension for Raycast

Create Daytona sandboxes and run code directly from Raycast.

## Features

- Create sandboxes
- Run code in sandboxes
- Create snapshots from base images
- Manage sandboxes
- Open web terminal
- Go to dashboard
- Read documentation

## Quick Start

### 1. Get API key

1. Go to [Daytona Dashboard](https://app.daytona.io/)
2. Navigate to [API Keys](https://app.daytona.io/dashboard/keys)
3. Create a new API key

### 2. Set API key

1. Open Raycast
2. Go to **Extensions** → **Daytona**
3. Enter your API key

### 3. Start using Daytona

1. Search for **Daytona** in the command palette
2. Select the command you want to use

## Commands

- **Create Sandbox**: Create a new sandbox
- **Run Code**: Run code in a sandbox
- **Create Snapshot**: Create a snapshot from a base image
- **Instant Sandbox**: Create a default Daytona sandbox instantly
- **Web Terminal**: Open the web terminal for a sandbox
- **Manage Sandboxes**: View sandboxes and run lifecycle actions
- **Read Documentation**: Read documentation
- **Open Dashboard**: Go to dashboard
