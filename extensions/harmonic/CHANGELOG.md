# Harmonic Changelog

## [Harmonic] - 2026-05-05

- Search for notes by name (e.g. "A flat", "C#", "Bb")
- View frequencies across all octaves (0-8) with MIDI numbers
- Search for chords using lead sheet notation (e.g. "Cmaj7", "Dm7b5", "F#7#11")
- Autocomplete suggestions for chord types
- Play tones with configurable tone type (Warm, Pure Sine, Bright, Soft Decay)
- Adjustable duration (1s, 2s, 5s, or infinite)
- Stack multiple pitches simultaneously
- Stop all playback with Cmd+Shift+S
- Copy frequencies and note names to clipboard
