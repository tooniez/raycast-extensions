# SnapOCR Via Paddle Changelog

## [Initial Version] - 2026-03-19

- Layout-aware OCR powered by Baidu PaddleOCR via Baidu AIStudio
- Stronger recognition for complex Chinese documents, handwriting, vertical text, tables, and formulas
- Quick OCR for fast screenshot-to-clipboard extraction
- Preview OCR for review-first structured OCR inside Raycast
- Export OCR Markdown for converting PDFs and images into Markdown bundles with extracted image assets
- Structured Markdown output that preserves titles, paragraphs, tables, formulas, and document hierarchy
- Optional document orientation classification, unwarping, and chart recognition
- Fast Mode for quicker screenshot and image OCR on large inputs
