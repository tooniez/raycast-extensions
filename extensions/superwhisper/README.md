# superwhisper

Change modes, start and stop recordings, search history, copy or paste the most recent transcript, and open settings.
Choose AI Processed or Unprocessed transcript variants in command settings for history copy/paste commands.
Set a custom recording directory for Search History via command preferences.

Requires the installation of superwhisper. 
Go to [https://superwhisper.com](https://superwhisper.com) to download & use for free.
