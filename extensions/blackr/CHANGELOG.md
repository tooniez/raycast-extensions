# Blackr Changelog

## [1.1.0] - 2026-06-22

- Add support for Raycast on Windows.
- Add platform-specific overlays for macOS and Windows.

## [Initial Release] - 2026-06-09

- Initial release.
- Add the `Blackr` command for a temporary black cleaning screen.
- Add configurable cleaning duration with a 60 second default.
- Add a subtle `Exit` button for ending early.
- Press `Esc` to dismiss the cleaning screen early.
