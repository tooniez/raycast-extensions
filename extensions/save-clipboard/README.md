# Save Clipboard As

Save files that you have in your clipboard to your device.

## Features

- **Images** — Screenshots and image files are detected and saved as JPEG
- **Files** — Any file copied to the clipboard is saved with its original extension
- **Text** — Plain text content is saved as `.txt`
- **HTML** — Rich text / HTML content is saved as `.html`

## Preferences

| Preference | Description | Default |
|---|---|---|
| Default Save Location | The directory where files are saved by default | `~/Downloads` |
