# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-11

### Fixed
- Fixed TypeScript build errors in API utility and component files.
- Resolved `RequestInit` type mismatch with `node-fetch`.
- Added missing `trash` status to Post and Page parameter types.
- Fixed extension icon path in `package.json`.
- Corrected `Image.Mask.Circle` implementation for avatars.

### Added
- Initial release of WordPress Manager for Raycast.
- Manage Posts, Pages, Comments, and Users.
- Media Library browser and Plugin management.
- Quick Post functionality.
