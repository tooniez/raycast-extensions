# Duden - German Dictionary for Raycast

A Raycast extension for searching German words and getting comprehensive information from [Duden.de](https://duden.de), Germany's authoritative dictionary.

## Features

### **Smart Search**
- **Real-time search** as you type (minimum 3 characters)
- **Debounced queries** for optimal performance
- **Auto-jump to details** when only one result is found
- **Limited results** (12 max) for clean interface
