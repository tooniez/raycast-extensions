# Changelog

All notable changes to this extension will be documented in this file.

## [Initial Release] - 2026-05-29

- Continuous mouse jiggling with start/stop commands
- Dramatic zigzag movements (80–150 px leaps every 2 seconds), bouncing off screen edges
- Zero dependencies — powered by macOS's built-in Swift interpreter
- Optional Fun Mode draws a visible orange trail overlay
