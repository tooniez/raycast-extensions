# Changelog

## [Initial Release] - 2026-05-05

- **Search CoCart Docs** — Full-text search across 120+ pages of CoCart API documentation with inline detail preview
- **Browse API Endpoints** — Browse endpoints by version (v2/v1) and category (Cart, Products, Sessions, Store, User, JWT, Plugins)
- **Search Hooks & Filters** — Individual lookup for every action hook, filter, and function with descriptions, usage examples, and copy code support
- **Search Error Codes** — Individual error code lookup with HTTP status filtering, color-coded by status code
- **Recent CoCart Docs** — Tracks recently opened pages across all commands with relative timestamps and source filtering
- **What's New in CoCart** — View latest updates and breaking changes
- **CoCart Quick Links** — Quick access to documentation, GitHub, and community resources
- **CoCart Docs Menu Bar** — Pin recently viewed docs to the menu bar (macOS)
- Local caching with 1-hour TTL and manual refresh via `Cmd+R`
