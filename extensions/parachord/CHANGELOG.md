# Changelog

## [Initial Version] - 2026-03-31

- Added Play/Pause, Next Track, and Previous Track controls
- Added Search & Play with smart "Artist - Track" and "Track by Artist" parsing
- Added Add to Queue for queuing tracks without interrupting playback
- Added AI DJ Chat with 8 quick prompt presets for music recommendations
- Added Open Artist command for quick artist page navigation
- Added Volume Control with visual list UI (Mute through 100%)
- Added Toggle Shuffle with persistent local state tracking
- Added Clear Queue with destructive action confirmation dialog
- Added Collection Radio for shuffling entire music library
- Added Open Parachord launcher command
