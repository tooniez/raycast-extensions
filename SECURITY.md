# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues or pull requests.**

If you believe you have found a security vulnerability, we encourage you to let us know right away. We will investigate all legitimate reports and do our best to quickly fix the problem.

Please report any vulnerabilities to **<security@raycast.com>** with the subject line `[Extension Security] <extension-name>`.

## Vulnerability Management

Raycast performs automated vulnerability scanning on this repository. Identified vulnerabilities are triaged and addressed by the Raycast team based on severity.
